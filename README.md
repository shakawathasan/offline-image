# Image â†’ Data URL Generator

A lightweight web tool that compresses images and converts them into Base64 Data URLs directly in your browser. Everything runs locally, so your files never leave your device.

## Features

* Convert images to Data URLs instantly
* Resize images using a max width/height limit
* Adjustable JPEG quality compression
* Copy or select generated output easily
* Mobile-friendly responsive interface
* Fully offline, no libraries or frameworks required

## How It Works

The app uses the HTML Canvas API to:

1. Load your selected image
2. Scale it based on the specified size limit
3. Compress it using JPEG quality settings
4. Output a Base64 Data URL ready to use in HTML, CSS, or JavaScript

## Usage

1. Clone or download this repository
2. Open `index.html` in your browser
3. Select an image
4. Adjust size and quality settings
5. Generate and copy your Data URL

## Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript
* Canvas API

## Privacy

All processing happens locally in your browser.
No uploads, tracking, or external services.
